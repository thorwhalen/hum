hum.utils.infinite_waveform¶

Simple infinite waveform

class hum.utils.infinite_waveform.InfiniteWaveform(gen_spectrum, noise_amp=None)[source]¶

A tiny little class emulating an infinite waveform with a given spectrum. Note that if choosing to add noise to the waveform, the generator will be considerably slower

query(bt, tt)[source]¶

Returns a generator of the infinite waveform from bt to tt

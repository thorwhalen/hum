hum.utils.librosa_utils¶

A few utils (specshow, melspectrogram) vendored from librosa.

This code was copied from parts of librosa, and adapted, so as to be able to use targeted functionality with less dependencies and manual installation (namely for libsndfile) than librosa has.

Librosa can be found here: https://librosa.org/

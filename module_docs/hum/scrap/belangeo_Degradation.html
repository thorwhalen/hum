hum.scrap.belangeo_Degradation

Template for a RadioPyo song (version 1.0).

A RadioPyo song is a musical python script using the python-pyo module to create the audio processing chain. You can connect to the radio here : http://radiopyo.acaia.ca/

There is only a few rules:

1 - It must be a one-page script. 2 - No soundfile, only synthesis. 3 - The script must be finite in time, with fade-in and fade-out

to avoid clicks between pieces. Use the DURATION variable.

belangeo - 2014

hum.scrap.belangeo_Degradation.dict_diff(a, b)[source]

Return the elements of a that are not keys of b.

>>> dict_diff({'a': 1, 'b': 2}, {'b': 3})
{'a': 1}

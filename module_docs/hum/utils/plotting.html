hum.utils.plotting.disp_wf(wf, sr=44100, autoplay=False, wf_plot_func=<function plot_wf>)[source]

Display waveform in Jupyter notebook

Parameters:

  • wf (array-like) – Waveform to display

  • sr (int, optional) – Sample rate of waveform, by default 44100

  • autoplay (bool, optional) – Whether to autoplay the audio, by default False

  • wf_plot_func (function, optional) – Function to plot the waveform, by default plot_wf (other example: plt.specgram)

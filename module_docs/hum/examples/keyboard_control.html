hum.examples.keyboard_control

Keyboard event reader that allows detecting keypress events. Run this script directly in a terminal.

hum.examples.keyboard_control.keyboard_reader(read_rate: float = 0.1, exit_key: str = 'escape', callback: Callable | str | None = None, arg_mapping: Dict | str | None = None, debug: bool = False) → Generator[Dict[str, Any] | None, None, None][source]

Generator that yields keyboard events at specified intervals.

Parameters:

  • read_rate – Time in seconds between checks. Defaults to 0.1.

  • exit_key – Key that will trigger exit when using the default example. Defaults to “escape”.

  • callback – Optional callback function to call when keys are pressed. Can be a callable or a string path to a callable in the format “module.path:function_name”.

  • arg_mapping – Optional mapping from raw keys to callback arguments. Can be a dict, a path to a JSON file, or a JSON string.

  • debug – Enable debug output for troubleshooting sound issues.

Yields:

Dict with key event information or None if no key was pressed

Example

>>> # Usage example (not a doctest)
>>> reader = keyboard_reader()
>>> for event in reader:
...     if event:
...         print(f"Key pressed: {event['key']}")
...     if event and event['key'] == 'escape':
...         break
